Synchronize Events from Sala Riviera

This will fetch and synchronize events from the Sala Riviera website (salariviera.com).

The synchronization process will:

  • Create the Sala Riviera venue if it doesn't exist
  • Fetch all upcoming events from the website
  • Create new events or update existing ones
  • Download event images and generate thumbnails
  • Create artists based on event titles
This process may take a few minutes depending on the number of events.
